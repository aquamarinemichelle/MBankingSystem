A full-featured online banking system built with Java Servlets, JSP, and MySQL. Provides secure banking operations including account management, deposits, withdrawals, transfers, and transaction history.

#Features
1.Authentication & Security
User Registration with auto-generated account numbers
Secure Login/Logout with session management
Password protection (hashed in production-ready implementations)

2.Banking Operations
Deposit Money with validation and limits
Withdraw Money with balance checking
Transfer Funds between accounts with transaction fees
Real-time Balance Updates

3.Account Management
Dashboard with account summary
Transaction History with filtering by date/type
Statement Generation with PDF/CSV export
Receipt Printing for all transactions

4.User Experience
Modern, Responsive UI with gradient designs
South African Rand (ZAR) currency formatting
Visual transaction flows with animations
Quick amount buttons for common transactions

5.Quick Start Prerequisites
Java JDK 17 or higher
Apache Tomcat 10.x
MySQL 8.0+
IDE (Eclipse/IntelliJ/VS Code)
